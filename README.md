## 前言

在数字化校园的大背景下，基于微信小程序的体育课评分系统应运而生。该系统旨在方便学生和教师进行体育课程评分，提高评分效率和准确性。本项目使用SSM框架，结合微信小程序，实现了一套简单易用、功能完善的体育课评分系统。

## 内容介绍

本项目主要包括以下功能模块：学生信息管理、课程信息管理、评分标准管理、评分记录管理等。通过微信小程序，学生可以实时查看课程评分，教师可以便捷地录入和修改评分。系统后端采用Java语言，结合Spring、SpringMVC、MyBatis等框架，确保系统的高效稳定运行。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
